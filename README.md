# paperspeaker
Text to speech App with React + Electron.

## Developmental Setup
- macOS (Sonoma)
- Node.js v20.9.0

For Text to speech, OpenAI TTS API is used (API key needs to save in `.env`).

Install & start.
```sh
npm install
npm start
```

While running on local server, start app with Electron.
```sh
npm run electron-start
```
